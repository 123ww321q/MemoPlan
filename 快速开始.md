# MemoPlan 快速开始指南

## 环境准备

### 1. 安装 Node.js
- 下载并安装 Node.js 18+ (https://nodejs.org/)
- 验证安装：`node --version`

### 2. 安装 Rust
- 下载并安装 Rust (https://www.rust-lang.org/tools/install)
- Windows 用户运行：`rustup-init.exe`
- 验证安装：`rustc --version`

### 3. 安装 Tauri CLI
```bash
npm install -g @tauri-apps/cli
```

## 开发步骤

### 1. 安装项目依赖
```bash
npm install
```

### 2. 启动开发模式
```bash
npm run tauri:dev
```

这会同时启动：
- Vite 开发服务器（前端）
- Tauri 应用窗口（桌面应用）

### 3. 开发调试
- 前端代码修改会自动热重载
- Rust 代码修改需要重启开发服务器
- 使用 F12 打开开发者工具

## 打包发布

### 构建 Windows 安装包
```bash
npm run tauri:build
```

生成的文件位置：
- `src-tauri/target/release/bundle/msi/MemoPlan_1.0.0_x64_zh-CN.msi`
- `src-tauri/target/release/bundle/nsis/MemoPlan_1.0.0_x64-setup.exe`

### 安装包说明
- `.msi` - Windows Installer 格式，适合企业部署
- `.exe` (NSIS) - 现代安装向导，推荐个人用户使用

## 常见问题

### Q: 开发模式启动失败？
A: 检查端口 5173 是否被占用，或修改 `vite.config.ts` 中的端口

### Q: 打包失败？
A: 确保 Rust 和 Node.js 环境正确安装，删除 `node_modules` 和 `src-tauri/target` 后重试

### Q: 如何修改应用图标？
A: 替换 `src-tauri/icons/` 目录下的图标文件

### Q: 如何添加新的 Tauri 命令？
A: 在 `src-tauri/src/main.rs` 中定义命令，然后在前端通过 `invoke()` 调用
